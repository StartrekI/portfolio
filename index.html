Featured Work

A collection of projects that showcase my skills and passion for building innovative solutions.

NutriSight — Multi-Modal Food Label Analyzer

Multi-modal Android platform combining CameraX, Google Cloud Vision OCR, and Claude LLM for real-time food label analysis at sub-10s p50 latency. SHA-256 Room SQLite cache cuts API calls ~70% and warm-path latency to <100ms. Implements NOVA-4 ultra-processed classification and Nutri-Score 2024 over a 500+ Indian product dataset (92% agreement). Serverless Node.js backend on Vercel Edge + Neon PostgreSQL with SSE streaming for token-level Claude output.

Android CameraX Cloud Vision Claude LLM Python Node.js PostgreSQL Vercel

HobbyHub — Activity & Hobby Discovery Platform

Full-stack platform with 29 REST endpoints over an 18-model Prisma schema (12 composite B-tree indexes) covering activities, gigs, trips, skills, and ideas. Two-stage geo search (SQL bounding-box prefilter + in-memory Haversine) fanned across 5 content types via parallel queries, plus a weighted matching engine (interest + proximity). Socket.io gateway with per-activity chat rooms, GPS throttling cutting presence writes ~80%, and stale-while-revalidate edge caching reducing DB hits ~70%.

Node.js Prisma PostgreSQL Socket.io Google OAuth REST APIs

TaskFlow — Real-Time Project Management

Full-stack project management tool with 20+ REST endpoints over a 5-table PostgreSQL schema (8 composite + partial indexes, FK cascades, JSONB, soft-delete), secured with JWT (bcrypt cost-12) and Zod validation. SSE pub-sub fan-out across per-project and per-user channels delivers updates in <100ms. 3-service Docker Compose stack (Nginx SPA + Express API + Postgres 16) with multi-stage builds and a React 18 + dnd-kit kanban frontend.

Node.js Express PostgreSQL React Docker JWT SSE
